### GodotTresEditor
A simple tool for edit .tres files extracted by [dsdecomp](https://github.com/GDRETools/gdsdecomp) 

Supported types:
- [x] OptimizedTranslation (.translation files)
- [x] FontFile (.fontdata files)
- [x] Texrutes (.ctex or .stex files)

Tool works if you convert binary files to text using [dsdecomp](https://github.com/GDRETools/gdsdecomp)
